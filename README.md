# 🚀 Minerador Viral Pro - Intelligence & Trends

O **Minerador Viral Pro** é uma solução avançada de inteligência de mercado desenvolvida para empreendedores de Dropshipping, E-commerce e Social Commerce. O objetivo central é eliminar o "achismo" e basear a escolha de produtos em dados reais de consumo no Brasil.

## 💎 Potencial Comercial e Diferenciais

Este aplicativo foi estruturado para ser uma ferramenta indispensável para vendedores que buscam alta conversão. Seus principais pilares de valor são:

* **Análise Multicanal em Tempo Real:** Diferente de listas estáticas, o app consulta o comportamento de busca do brasileiro (Google Trends) e a oferta dos maiores marketplaces (Mercado Livre) no momento do clique.
* **Foco em Escala (Dropshipping):** Identifica produtos "virais" antes que o mercado fique saturado, permitindo uma margem de lucro maior.
* **Acessibilidade Total (Cloud Native):** Desenvolvido sobre Streamlit, o app é 100% responsivo, funcionando perfeitamente em smartphones, tablets e desktops sem necessidade de instalação.
* **Baixo Custo de Manutenção:** Arquitetura limpa e eficiente, utilizando APIs e Raspagem de Dados de alto desempenho.

## 🛠️ Funcionalidades Atuais

1.  **Módulo Google Brasil:** Extrai os termos mais pesquisados nos últimos 7 dias, revelando a intenção de compra real do consumidor.
2.  **Módulo Marketplace Crawler:** Varre os anúncios mais relevantes do Mercado Livre para validar preços e concorrência.
3.  **Filtro por Nicho:** Permite uma pesquisa segmentada (ex: Pet Shop, Beleza, Cozinha Inteligente).
4.  **Interface Intuitiva:** Design focado em UX (User Experience) para agilizar a tomada de decisão.
